# newCommerceApi

O projeto <b>newCommerceApi</b> disponibiliza uma api REST, que permite o acesso a vários serviços:

* Address
* Category
* Company
* Institucional
* itemOrder
* Order
* Payment
* Printer
* Product
* Slide
* SubCategory
* Supplier
* User
* Variation

>Novos models/services e controllers sendo escritos

>Authenticação via JWT
